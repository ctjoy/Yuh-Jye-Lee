發照單位分級方式


證照介紹




IC3

IC3是Internet and Computing Core Certifications的縮寫，中文譯名為網際網路與資訊核心能力認證。

美國微軟全球認證中心思遞波公司，是微軟旗下的IT認證專業機構。




MOS

「微軟 Office 專家 (MOS)」完全展現的進階 Microsoft 桌上型軟體使用能力，並受到全球一致認同。




MCAD

「微軟認證應用程式開發工程師 (MCAD)」代表專精在使用 Microsoft 技術來開發及維護部門層次的應用程式，

元件，Web 或桌上型電腦用戶端，或是後端資料服務。




MCSD

「微軟認證解決方案開發工程師 (MCSD)」代表專精在使用 Microsoft 開發工具，技術，平台與 Windows 作業系統

來設計及開發尖端企業解決方案。




MCPD

「微軟認證程式開發專家 (MCPD)」認證完全展現IT專家，在設計、開發，以及部署針對特殊工作角色的應用程式所需具備的多樣技能。

這些認證代表代表您具備成功執行工作的所需技能。




MCSA

「微軟認證系統管理師 (MCSA)」代表專精在管理以 Microsoft Windows 作業系統為基礎的網路與系統環境上。

特別涵蓋 「MCSA：Messaging」 以及 「MCSA：Security」。




MCSE

「微軟認證系統工程師 (MCSE)」代表專精在針對以 Windows 作業系統與 Windows Server 軟體為基礎的設計與實作基礎結構解決方案上，

特別涵蓋 「MCSE：Messaging」以及「MCSE：Security」。




MCTS

「微軟認證技術專家 (MCTS) 」認證能夠讓專業人員將專注在特定的技術目標上，並透過在他們工作中擅長使用的技術，

展示他們薈萃的知識及獨到的專長，以突顯出自身的能力。




MCITP

「微軟認證 IT 開發專家 (MCITP)」認證可完全展示您在規劃、部署、支援、維護和最佳化 IT 基礎架構上的特殊專長。




MCA

微軟認證架構師（MCA）
