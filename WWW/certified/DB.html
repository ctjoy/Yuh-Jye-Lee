證照介紹


Microsoft Certified Database Administrator Microsoft Certified Database Administrator （ MCDBA )

•微軟DBA認證為微軟針對SQL Server資料庫所推出的獨立認證，特別針對 資料庫管理員所設計，持有此認證的專業人員，

可被視為具有管理SQL Server以及設計資料庫應用的專業能力與知識。

•本認證已隨著 Exam 70-228 與 Exam 70-229 於2009年3月31日停考後正式走入歷史。

•由於微軟新一代認證階層的推展，MCDBA 已由 MCITP 取代，但現行的 MCDBA on SQL Server 2000 仍為永久有效。
