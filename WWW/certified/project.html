•考試科目: 通過310-055 SCJP 5.0或310-065 SCJP 6.0其中一科考試科目

4.物件導向程式設計

5.物件、變數的生命期與列舉類型

7.資料的輸入輸出

8.數字與文字相關類別

9.格式設定相關類別

10.集合與泛型

11.多執行緒程式設計

12.繼承、多型、封裝




OCJD(SCJD)

•簡介:

進一步測試Java 開發應用程序的能力，必須先取得OCJP認證資格者，才可參加本項認證測驗。

考試者必須先完成一個程序的設計方案(實作題)，再回答與此方案相關的一些問題。

• 考試科目: 310-027

• 報考資格: 通過OCJP證照認證。

•認證方式:

第一階段是程式設計實作，您必須下載程式並且完成該程式。

第二階段是編號310-027的考試 ，將會針對您第一階段的程式設計出題，一般有5~10題簡答申論題，以英文做答。

• 通過標準: 第一階段答對80%以上通過；第二階段答對70%以上通過。

• 考試重點:

1.分析、設計、實現和測試“自底而上”的intranet的商業應用

2.理解JDBC和編寫與SQL數據庫連接的程序

3.解釋軟件開發周期，每階段的目標，可使用適當的Java程序類型，了解API設計的基本元素，根據項目中的功能需求確定程序組件

4.描述關系數據庫的設計及其設計原則

5.了解JDBC及其使用

6.使用良好的GUI設計原則設計用戶界面

7.可以使用Socket連接以適當格式傳輸數據，並將其集成到應用程序中

8.描述一層，兩層及多層數據庫結構及相關問題，創建一個多層Java小程序或應用程序

9.了解在Solaris和Windows平臺之間的移植問題

•簡介:

•考試科目: 310-083

•報考資格: 通過OCJP證照認證。

•認證方式: 電腦上機，單選、複選及拖曳題共69題。

•通過標準: 答對72%以上通過。

•考試重點:

1.The Servlet Model

2.The Structure and Deployment of Modern Servlet Web Applications

3.The Servlet Container Model

4.Designing and Developing Servlets to Handle Server-side Exceptions

5.Designing and Developing Servlets Using Session Management

6.Designing and Developing Secure Web Applications

7.Designing and Developing Thread-safe Servlets

8.The JavaServer Pages Technology Model

9.Designing and Developing Reusable Web Components

10.Designing and Developing JSP Pages Using JavaBeans Components

11.Designing and developing JSP Pages Using Custom Tags

12.Designing and Developing a Custom Tag Library

13.Design Patterns




OCBCD (SCBCD)，商業元件

•簡介:

本認證是為負責設計、開發、測試、部署及整合EJB應用程式之Enterprise JavaBeans(EJB)技術商業元件開發人員所提供的。

本項認證是針對已取得SCJP認證的Java程式設計人員，在Java EE平台技術中，其必須專精於封裝商業邏輯的伺服端分散式元件應用程式。

•考試科目: 310-091

•報考資格: 通過OCJP證照認證。

•認證方式: 電腦上機，單選、複選及拖曳題共61題。

•通過標準: 答對59%以上通過。

•考試重點:

1.Basic EJB

2.EJB Architecture

3.Exposing Yourself

4.Being a Session Bean

5.Entities are Persistant

6.Being an Entity Bean

7.When Beans Relate

8.Getting the Message

9.The Atomic Age

10.When Beans Go Bad

11.Protect Your Secrets

12.The Joy of Deployment

OCEA (SCEA) (SCAJ)

•簡介:專為負責建構及設計具可擴充性、彈性及高安全性之Java 2 Platform的企業架構設計師所提供。

•考試科目: 310-052

•報考資格: 無

•認證方式:

[PART I] Knowledge-Based Exam:題目包括複選題、簡答題共48題,測驗時間為75分鐘。本測驗之及格標準為至少答對68%的題目。

[PART II] Assignment:系統設計專題，及格標準為70%。

[PART III] Essay Exam:在90分鐘內回答四個問答題。本測驗與系統設計專題測驗一併計分。

•通過標準:Part1~3皆通過。

•考試重點:

1.一般架構(Common Architectures)

2.舊有專屬系統之連接(Legacy Connectivity)

3.Enterprise JavaBeans

4.Enterprise JavaBeans Container模型

5.通訊協定

6.J2EE技術的適用性

7.Design Pattern

8.訊息傳遞(Messaging)

9.國際化(Internationalization)

10.安全性
